## Questions

### 1. Basic Info

**1.1 What's your name?**
- Type: Short text
- Required: Yes

**1.2 Email address**
- Type: Email
- Required: Yes

**1.3 Where are you based?**
- Type: Short text
- Placeholder: "City, Country"
- Required: Yes

**1.4 What's your timezone?**
- Type: Dropdown
- Options: Major timezones (UTC-8 to UTC+12)
- Required: Yes

---

### 2. Current Situation

**2.1 What's your current situation?**
- Type: Multiple choice
- Required: Yes
- Options:
  - Employed full-time, building on the side
  - Freelancer/contractor
  - Full-time on my own projects
  - Student
  - Between things, looking for what's next
  - Other

**2.2 If "Other," please explain:**
- Type: Short text
- Required: No
- Show if: "Other" selected

---

### 3. Skills & Experience

**3.1 What have you built?**
- Type: Long text
- Required: Yes
- Placeholder: "Links to projects, GitHub, portfolio - anything that shows you can ship. If no links, describe what you've built."

**3.2 What tools/tech are you comfortable with?**
- Type: Checkboxes (select all that apply)
- Required: Yes
- Options:
  - JavaScript/TypeScript
  - React/Next.js
  - Python
  - n8n / Make / Zapier
  - AI/LLM APIs (OpenAI, Claude, etc.)
  - Databases (SQL, Convex, Supabase, etc.)
  - No-code tools (Webflow, Bubble, etc.)
  - Other

**3.3 If "Other" tools, please list:**
- Type: Short text
- Required: No
- Show if: "Other" selected

**3.4 How would you rate your current skill level?**
- Type: Multiple choice
- Required: Yes
- Options:
  - Beginner (still learning fundamentals)
  - Intermediate (can build things with some guidance)
  - Advanced (can build most things independently)
  - Expert (can architect complex systems)
